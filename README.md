# Passaggi operativi

### 1. Compilazione
```bash
ant -f Tablut/build.xml compile
```

### 2. Avvio del Server
```bash
ant -f Tablut/build.xml gui-server
```

### 3. Avvio dell'Agente (Team Lucani)
```bash
./runmyplayer.sh [COLORE] [TIMER] [ip]
```
